# Diogo Oliveira

### How to make complexity approachable.

That's the thread running through everything below. Different mediums, different problems — story, systems, documentation, an interactive prototype — but the same instinct applied each time: take something dense, and find the exact amount of surface a person can hold at once, so the complexity underneath never feels like a wall.
